# Beginner Coders Guide

This project is a beginner-friendly website designed to introduce new coders to basic programming languages and coding essentials.

## Features
- Learn about basic coding languages:
  - HTML
  - CSS
  - JavaScript
  - Python
  - Java
  - Scratch
- Interactive buttons that display information about each language
- A second page with beginner coding essentials and helpful resources
- Simple, clean design with consistent styling

## Files Included
- `index.html` – main homepage
- `coding-essentials.html` – second page with coding tools and resources
- `styles.css` – styling for the website
- `script.js` – interactive JavaScript for language information
- `README.md` – project description

## Purpose
The purpose of this website is to help beginners feel more comfortable starting their coding journey by giving them simple explanations and useful resources.
